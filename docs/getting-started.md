# Getting Started

Previous: README  
Next: [Configuration Model](configuration-model.md)

## Prerequisites

- Node.js LTS
- `pnpm`

## Install

```bash
pnpm install
```

## Minimal setup

```bash
cp .env.example .env
```

## Run locally

```bash
pnpm dev
```

## Validate and build

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm preview
```
